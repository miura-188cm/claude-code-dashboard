# .claude/teams/{team-name}/inboxes/{agent}.json

チーム内のエージェント間メッセージ通信。各エージェントごとに受信ボックスが存在する。

---

```json
[
  {
    "from": "<sender-agent-name>",
    "text": "<message-body>",
    "summary": "<message-summary>",
    "timestamp": "<ISO8601>",
    "read": false
  }
]
```

## フィールド説明

| フィールド  | 型      | 説明                                    |
| ----------- | ------- | --------------------------------------- |
| `from`      | string  | 送信元エージェント名（例: `team-lead`） |
| `text`      | string  | メッセージ本文                          |
| `summary`   | string  | メッセージの要約                        |
| `timestamp` | string  | 送信日時（ISO8601形式）                 |
| `read`      | boolean | 既読フラグ                              |

## 備考

- ファイル名の `{agent}` 部分が受信者のエージェント名（例: `planner.json`, `explorer-note-update.json`）
- 配列の各要素が1つのメッセージ
