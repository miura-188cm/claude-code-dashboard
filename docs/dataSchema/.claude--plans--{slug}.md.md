# .claude/plans/{slug}.md

実装計画ドキュメント。EnterPlanMode で作成される Markdown ファイル。

---

## ファイル名パターン

```
<adjective>-<verb>-<noun>.md
```

例: `ancient-painting-pebble.md`, `crispy-swimming-rain.md`

## フォーマット

Markdown形式のフリーテキスト。構造は計画内容に依存する。典型的には以下を含む:

```markdown
# <計画タイトル>

## Context

<背景・制約の説明>

## 方針

<実装方針>

## 実装ステップ

### Step 1: <ステップ名>

<詳細>

### Step 2: <ステップ名>

<詳細>
```

## 備考

- 構造化されたJSONではなく、自由形式のMarkdown
- ファイル名のslugはランダム生成（セッションslugとは別）
- 内容のパースには自然言語処理またはセクション分割が必要
