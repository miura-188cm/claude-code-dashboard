# .claude/settings.json

ユーザーのグローバル設定。権限ルール、言語設定、環境変数等を管理する。

---

```json
{
  "permissions": {
    "allow": ["<permission-rule>"]
  },
  "language": "<language-code>",
  "autoUpdatesChannel": "<stable | beta>",
  "feedbackSurveyState": {
    "lastShownTime": 0
  },
  "env": {
    "<ENV_VAR_NAME>": "<value>"
  }
}
```

## フィールド説明

| フィールド                          | 型       | 説明                                                                                                   |
| ----------------------------------- | -------- | ------------------------------------------------------------------------------------------------------ |
| `permissions.allow`                 | string[] | 許可済みツール・コマンドのルール一覧。`Bash(command:*)`, `Read(path)`, `WebFetch(domain:xxx)` 等の形式 |
| `language`                          | string   | 応答言語設定（`japanese` 等）                                                                          |
| `autoUpdatesChannel`                | string   | 自動更新チャンネル                                                                                     |
| `feedbackSurveyState.lastShownTime` | number   | フィードバック調査を最後に表示した時刻（epoch ms）                                                     |
| `env`                               | object   | 環境変数のオーバーライド                                                                               |
