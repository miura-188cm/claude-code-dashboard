# .claude/projects/{project-id}/{session-id}.jsonl

セッションの全メッセージ・ツール呼び出し・token使用量を記録するJSONLファイル。
**最も情報量が多い最重要データソース。**

- `{project-id}`: プロジェクトパスを `-` 区切りにしたもの（例: `-Users-tm-Documents-project-foo`）
- `{session-id}`: UUID v4 形式のセッションID
- 各行が1つのJSONオブジェクト。`type` フィールドで種別を判別する。

---

## type: "user"

ユーザーからのメッセージ。

```json
{
  "type": "user",
  "uuid": "<message-uuid>",
  "parentUuid": "<parent-message-uuid | null>",
  "sessionId": "<session-uuid>",
  "timestamp": "<ISO8601>",
  "userType": "external",
  "isSidechain": false,
  "cwd": "<working-directory-path>",
  "gitBranch": "<branch-name>",
  "version": "<claude-code-version>",
  "permissionMode": "<permission-mode>",
  "message": {
    "role": "user",
    "content": "<text or content-blocks>"
  }
}
```

- `message.content` は文字列、または以下のブロック配列:

```json
[
  { "type": "text", "text": "<user-input-text>" },
  { "type": "tool_result", "tool_use_id": "<tool-use-id>", "content": "<result>" }
]
```

## type: "assistant"

アシスタントからの応答。**token使用量はここに含まれる。**

```json
{
  "type": "assistant",
  "uuid": "<message-uuid>",
  "parentUuid": "<parent-message-uuid>",
  "sessionId": "<session-uuid>",
  "timestamp": "<ISO8601>",
  "userType": "external",
  "isSidechain": false,
  "cwd": "<working-directory-path>",
  "gitBranch": "<branch-name>",
  "version": "<claude-code-version>",
  "slug": "<session-slug>",
  "requestId": "<api-request-id>",
  "message": {
    "role": "assistant",
    "content": [
      { "type": "text", "text": "<response-text>" },
      {
        "type": "tool_use",
        "id": "<tool-use-id>",
        "name": "<tool-name>",
        "input": {}
      }
    ],
    "usage": {
      "input_tokens": 0,
      "output_tokens": 0,
      "cache_creation_input_tokens": 0,
      "cache_read_input_tokens": 0,
      "cache_creation": {
        "ephemeral_1h_input_tokens": 0,
        "ephemeral_5m_input_tokens": 0
      },
      "server_tool_use": {
        "web_search_requests": 0,
        "web_fetch_requests": 0
      },
      "service_tier": "<tier>",
      "speed": "<speed>",
      "inference_geo": "<geo>"
    }
  }
}
```

### tool_use ブロックの例（ツール別）

#### Read

```json
{
  "type": "tool_use",
  "id": "<tool-use-id>",
  "name": "Read",
  "input": {
    "file_path": "<absolute-file-path>",
    "offset": 0,
    "limit": 0
  }
}
```

#### Write

```json
{
  "type": "tool_use",
  "id": "<tool-use-id>",
  "name": "Write",
  "input": {
    "file_path": "<absolute-file-path>",
    "content": "<file-content>"
  }
}
```

#### Edit

```json
{
  "type": "tool_use",
  "id": "<tool-use-id>",
  "name": "Edit",
  "input": {
    "file_path": "<absolute-file-path>",
    "old_string": "<string-to-replace>",
    "new_string": "<replacement-string>"
  }
}
```

#### Bash

```json
{
  "type": "tool_use",

  "name": "Bash",
  "input": {
    "command": "<shell-command>",
    "description": "<command-description>",
    "timeout": 0
  }
}
```

#### Glob

```json
  "type": "tool_use",
```

```json

  "type": "user",
  "uuid": "<message-uuid>",
  "parentUuid": "<parent-message-uuid>",
  "sessionId": "<session-uuid>",
  "timestamp": "<ISO8601>",
  "userType": "external",
  "isSidechain": false,
  "cwd": "<working-directory-path>",
  "gitBranch": "<branch-name>",
  "version": "<claude-code-version>",
  "slug": "<session-slug>",
  "toolUseResult": {
    "success": true,
    "output": "<result-content>"
  },
  "sourceToolAssistantUUID": "<assistant-message-uuid>",

    "role": "user",

      {
```

## type: "system"

システムイベント。`subtype` で分類される。

### subtype: "compact_boundary"

コンテキスト圧縮イベント。

```json
{
  "type": "system",
  "subtype": "compact_boundary",
  "uuid": "<message-uuid>",
  "parentUuid": null,
  "logicalParentUuid": "<logical-parent-uuid>",
  "sessionId": "<session-uuid>",
  "timestamp": "<ISO8601>",
  "userType": "external",
  "isSidechain": false,
  "cwd": "<working-directory-path>",
  "gitBranch": "<branch-name>",
  "version": "<claude-code-version>",
  "slug": "<session-slug>",
  "content": "Conversation compacted",
  "isMeta": false,
  "level": "info",
  "compactMetadata": {
    "trigger": "<manual | auto>",
    "preTokens": 0
  }
}
```

### subtype: "turn_duration"

ターンの所要時間。

```json
{
  "type": "system",
  "subtype": "turn_duration",
  "uuid": "<message-uuid>",
  "parentUuid": "<parent-message-uuid>",
  "sessionId": "<session-uuid>",

  "slug": "<session-slug>",

```

```json
{
  "type": "progress",

  "sessionId": "<session-uuid>",
  "timestamp": "<ISO8601>",

  "cwd": "<working-directory-path>",
  "gitBranch": "<branch-name>",
  "version": "<claude-code-version>",
  "slug": "<session-slug>",
  "data": {
    "type": "hook_progress",
    "hookEvent": "<PreToolUse | SessionStart | SessionEnd>",
    "hookName": "<hook-identifier>"
  },
  "toolUseID": "<tool-use-id>",
  "parentToolUseID": "<tool-use-id>"
}
```

## type: "queue-operation"

セッションキューの操作。セッションの開始・終了を示す。

```json
{
  "type": "queue-operation",
  "operation": "<enqueue | dequeue>",
  "timestamp": "<ISO8601>",
  "sessionId": "<session-uuid>"
}
```

## type: "file-history-snapshot"

ファイル変更のスナップショット。undo/redo用。

```json
{
  "type": "file-history-snapshot",
  "messageId": "<message-uuid>",
  "isSnapshotUpdate": false,
  "snapshot": {
    "messageId": "<message-uuid>",
    "trackedFileBackups": {},
    "timestamp": "<ISO8601>"
  }
}
```

```json
  "timestamp": "<ISO8601>",
  "sessionId": "<session-uuid>"
```
