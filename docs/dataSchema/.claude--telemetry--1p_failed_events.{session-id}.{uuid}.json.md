# .claude/telemetry/1p_failed_events.{session-id}.{uuid}.json

送信に失敗したテレメトリイベント。Anthropicへの送信がエラーになった場合にローカルに保存される。

---

## ファイル名パターン

```
1p_failed_events.<session-uuid>.<event-uuid>.json
```

## 備考

- 送信失敗時のフォールバックとして保存される
- 正常に送信されたテレメトリはローカルに残らない
- ファイル数が多い場合、ネットワーク接続の問題を示唆する
- 内容は Anthropic のテレメトリスキーマに依存するため、分析対象としての優先度は低い
