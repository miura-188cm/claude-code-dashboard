# .claude/todos/{session-id}-agent-{agent-id}.json

セッション内でエージェントが管理するTodoリスト。TodoWriteツールで作成・更新される。

---

```json
[
  {
    "id": "<todo-number>",
    "content": "<todo-description>",
    "status": "<pending | in_progress | completed>"
  }
]
```

## フィールド説明

| フィールド | 型     | 説明                                                |
| ---------- | ------ | --------------------------------------------------- |
| `id`       | string | Todo番号（連番）                                    |
| `content`  | string | Todoの内容・説明                                    |
| `status`   | string | ステータス（`pending`, `in_progress`, `completed`） |

## 備考

- ファイル名の `{session-id}` と `{agent-id}` は同じUUIDの場合もある（メインエージェントの場合）
- 空の場合は `[]` が記録される
