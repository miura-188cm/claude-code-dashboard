# .claude/stats-cache.json

日次アクティビティの集計キャッシュ。Claude Code が内部的に計算・保持する統計サマリー。

---

```json
{
  "version": 0,
  "lastComputedDate": "<YYYY-MM-DD>",
  "dailyActivity": [
    {
      "date": "<YYYY-MM-DD>",
      "messageCount": 0,
      "sessionCount": 0,
      "toolCallCount": 0
    }
  ],
  "dailyModelTokens": [],
  "modelUsage": {},
  "totalSessions": 0,
  "totalMessages": 0,
  "longestSession": {
    "sessionId": "<session-uuid>",
    "duration": 0,
    "messageCount": 0,
    "timestamp": "<ISO8601>"
  },
  "firstSessionDate": "<ISO8601>",
  "hourCounts": {
    "<hour-0-to-23>": 0
  },
  "totalSpeculationTimeSavedMs": 0
}
```

## フィールド説明

| フィールド                    | 型     | 説明                                                   |
| ----------------------------- | ------ | ------------------------------------------------------ |
| `version`                     | number | スキーマバージョン                                     |
| `lastComputedDate`            | string | 最後に集計が実行された日付                             |
| `dailyActivity`               | array  | 日付ごとのメッセージ数・セッション数・ツール呼び出し数 |
| `dailyModelTokens`            | array  | 日付ごとのモデル別token使用量                          |
| `modelUsage`                  | object | モデル別の累計使用状況                                 |
| `totalSessions`               | number | 累計セッション数                                       |
| `totalMessages`               | number | 累計メッセージ数                                       |
| `longestSession`              | object | 最長セッションの情報                                   |
| `firstSessionDate`            | string | 初回セッションの日時                                   |
| `hourCounts`                  | object | 時間帯（0〜23）ごとの利用回数                          |
| `totalSpeculationTimeSavedMs` | number | 投機的実行で節約された時間（ミリ秒）                   |
