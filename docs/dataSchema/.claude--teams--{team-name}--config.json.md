# .claude/teams/{team-name}/config.json

チームの構成情報。チーム名、メンバー一覧、使用モデル、プロンプト等を保持する。

---

```json
{
  "name": "<team-name>",
  "description": "<team-purpose-description>",
  "createdAt": 0,
  "leadAgentId": "<agent-name>@<team-name>",
  "leadSessionId": "<session-uuid>",
  "members": [
    {
      "agentId": "<agent-name>@<team-name>",
      "name": "<agent-name>",
      "agentType": "<team-lead | Plan | Explore | general-purpose>",
      "model": "<claude-opus-4-6 | claude-sonnet-4-6 | inherit>",
      "prompt": "<agent-task-prompt>",
      "color": "<color-name>",
      "planModeRequired": false,
      "joinedAt": 0,
      "tmuxPaneId": "<pane-id | in-process>",
      "cwd": "<working-directory-path>",
      "subscriptions": [],
      "backendType": "<in-process>"
    }
  ]
}
```

## フィールド説明

### トップレベル

| フィールド      | 型     | 説明                                                |
| --------------- | ------ | --------------------------------------------------- |
| `name`          | string | チーム名（ディレクトリ名と一致）                    |
| `description`   | string | チームの目的・説明                                  |
| `createdAt`     | number | 作成日時（Unix epoch ミリ秒）                       |
| `leadAgentId`   | string | リーダーエージェントID（`{name}@{team-name}` 形式） |
| `leadSessionId` | string | リーダーのセッションID                              |
| `members`       | array  | メンバー一覧                                        |

### members[]

| フィールド         | 型      | 説明                                                        |
| ------------------ | ------- | ----------------------------------------------------------- |
| `agentId`          | string  | エージェントID（`{name}@{team-name}` 形式）                 |
| `name`             | string  | エージェント名（`team-lead`, `planner`, `explorer-xxx` 等） |
| `agentType`        | string  | エージェントタイプ                                          |
| `model`            | string  | 使用モデル。`inherit` は親から継承                          |
| `prompt`           | string  | エージェントに与えたプロンプト                              |
| `color`            | string  | 表示色                                                      |
| `planModeRequired` | boolean | プランモード強制か                                          |
| `joinedAt`         | number  | 参加日時（Unix epoch ミリ秒）                               |
| `tmuxPaneId`       | string  | tmuxペインID。`in-process` はインプロセス実行               |
| `cwd`              | string  | 作業ディレクトリ                                            |
| `subscriptions`    | array   | サブスクリプション                                          |
| `backendType`      | string  | バックエンドタイプ                                          |
