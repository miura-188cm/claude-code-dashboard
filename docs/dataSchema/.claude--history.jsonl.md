# .claude/history.jsonl

ユーザーが入力したコマンド・テキストの履歴。スラッシュコマンド（`/mcp`, `/clear` 等）も記録される。

---

```json
{
  "display": "<user-input-text>",
  "timestamp": 0,
  "project": "<project-directory-path>",
  "sessionId": "<session-uuid>",
  "pastedContents": {}
}
```

## フィールド説明

| フィールド       | 型     | 説明                                                                                      |
| ---------------- | ------ | ----------------------------------------------------------------------------------------- |
| `display`        | string | ユーザーが入力したテキスト。スラッシュコマンド（`/mcp`, `/clear` 等）やプロンプト文が入る |
| `timestamp`      | number | 入力時刻（Unix epoch ミリ秒）                                                             |
| `project`        | string | 入力時のプロジェクトディレクトリパス                                                      |
| `sessionId`      | string | セッションID（存在する場合。セッション開始前のコマンドにはない）                          |
| `pastedContents` | object | ペーストされたコンテンツ（ある場合）                                                      |

## 備考

- `sessionId` はセッション開始後のエントリにのみ付与される
- `display` の先頭がスラッシュ（`/`）で始まるものはスラッシュコマンド
